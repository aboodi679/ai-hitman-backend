// 03 — Backend

Live API Endpoints

LIVE BASE URL
https://4kbub5enwb.execute-api.us-east-1.amazonaws.com/production
LIVE
GET
/ping
Health check — verifies the API is alive and responding from Lambda.
Called by: Developer
POST
/upload-model
Uploads the trained RL zombie model file to AWS S3 cloud storage.
Called by: Hassan (post-training)
GET
/get-model-url
Returns a signed S3 URL so Unity can download the RL model on game start.
Called by: Unity (on start)
POST
/save-session
Saves player session stats — kills, health, weapon, time, AI type — to DynamoDB.
Called by: Unity (session end)
GET
/get-stats
Returns all player sessions from DynamoDB for the live dashboard.
Called by: Website
